List
<ul></ul>
<ol></ol> 		An unordered list starts with the <ul></ul> tag. Each list item starts with the < li >tag. The list items will be marked with bullets (small black circles) by default.
An ordered list starts with the <ol></ol> tag. Each list item starts with the < li >tag. The list items will be marked with numbers by default.
Image tag
<img src = "name" alt="text">		 The <img> tag has two required attributes:
src - Specifies the path to the image
alt - Specifies an alternate text for the image,
if the image for some reason cannot be displayed
